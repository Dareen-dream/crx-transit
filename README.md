## 功能列表

- 页面英文划词翻译
- 连续对多个单词进行划词翻译
- 对超链接中的文本进行划词翻译
- 调整页面划词翻译结果显示的时间长短
- 适应更多的页面，包括 Iframe 嵌套
- 支持在窗口边缘和选词附近两种方式显示翻译结果
- 支持百度和有道翻译两种翻译服务

## Install

	$ npm install

## Development

    npm run dev chrome
    npm run dev firefox
    npm run dev opera
    npm run dev edge

## Build

    npm run build chrome
    npm run build firefox
    npm run build opera
    npm run build edge

## Environment

The build tool also defines a variable named `process.env.NODE_ENV` in your scripts.
